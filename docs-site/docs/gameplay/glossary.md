# Game Glossary

Reference tables for the political and media landscape in Press Office.

## The Cabinet

| Position     | Default Name                         | Role                           |
| :----------- | :----------------------------------- | :----------------------------- |
| **State**    | Secretary of State                   | Diplomacy, Foreign Affairs     |
| **Defense**  | Secretary of Defense                 | Military, National Security    |
| **Treasury** | Secretary of the Treasury            | Economy, Budget                |
| **Justice**  | Attorney General                     | Law, Civil Rights              |
| **HHS**      | Secretary of Health & Human Services | Public Health, Social Programs |
| **Homeland** | Secretary of Homeland Security       | Borders, Domestic Safety       |
